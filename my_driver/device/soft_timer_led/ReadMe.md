
# soft_timer_led简介

## 项目: 软定时器, 内核时钟中断, 系统节拍计数器
```
理解内核定时器: 软件定时器, 实现"延时"，"周期性任务", "超时检测"

底层依赖内核时钟中断和jiffies(系统节拍计数器)实现计时
```


## 重点
1. **定时器**
    - 回调函数: 属于软中断上下文, 不能睡眠. 只能使用原子操作, 自旋锁
        - 考虑到定时器运行过程中, 设备可能被删除, 此时可以在回调函数中检查设备状态, 从而重新启用定时器
    - timer_setup: 创建定时器
    - mod_timer: 其中的过期时间是绝对时间
    - del_timer_sync: 同步等待回调函数结束后, 从队列中删除

2. **自旋锁**
    - 适用于中断上下文.
3. **中断**
    - 中断处理程序的架构: 顶半部, 底半部
    - 中断编程方法
        - 申请和释放中断
        - 使能和屏蔽中断
        - 中断底半部tasklet
        - 工作队列
        - 软中断机制
        - thread_irq
    - 多个设备共享同一个中断号处理过程



# 模块简介
    - 定时器分类
        - 普通定时器
        - 高精度定时器: 基于硬件时钟
        - 延迟工作队列: 基于工作队列和定时器，回调函数在进程上下文执行
        - watchdog定时器: 硬/软结合, 监控系统运行
