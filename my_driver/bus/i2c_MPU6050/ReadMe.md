### 概念补充
1. **陀螺仪**
    - 功能: 测量和维持角速度的装置. 三轴陀螺仪是测量绕X, Y, Z三个轴旋转的角速度, 单位是"度/s"
    - 物质组成
        - MEMS(微机电系统)陀螺仪, 核心物质是单晶硅
    - 基本原理: **科里奥利效应**
        - 驱动: 通过静电力, 使一个微小的质量块在其谐振频率上, 沿一个方向(**驱动轴**)持续地来回振荡
        - 旋转输入: 当这个芯片绕**垂直于驱动轴的轴线**旋转时, 振荡的质量块会受到一个垂直于驱动轴和旋转的惯性力(科里奥利力)
        - 检测: 这个力会使质量块在第三个方向(称为**检测轴**)上产生微小的位移. 这个位移与输入的角速度成正比
        - 转换: 这个微小的位移通过电容检测转换为电信号, 经过放大、解调、滤波和波束转换, 最终输出与角速度成正比的数字信号
    - 种类
        - 机械转子陀螺仪
            - 原理: 高速旋转的角动量守恒原理
            - 特点: 精度极高, 但体积大、结构复杂、成本昂贵
        - 光纤陀螺仪(FOG)
            - 原理: 基于萨格纳克效应，两束光在旋转的光纤环中产生相位差
            - 特点: 精度高，无运动部件，可靠性高，动态范围大
        - 激光陀螺仪(RLG)
            - 原理: 基于萨格纳克效应，在环形激光谐振腔中产生频率差
            - 特点: ：精度极高，可靠性高，对温度变化不敏感，但存在“闭锁”效应（低转速下无输出），制造工艺复杂，成本极高
        - MEMS陀螺仪
            - 原理: 基于科里奥利效应的振动式微机械结构
            - 特点: 体积微小，功耗极低，成本极低，可批量生产，抗冲击能力强
        - 半球谐振陀螺仪
            - 原理: 利用半球壳谐振子在旋转时驻波进动效应
            - 特点: 极高精度、极高可靠性、极长寿命、无机械磨损，是战略级惯导的核心器件，但制造难度极大，成本极高

2. **加速度计**
    - 功能: 测量物体线加速度的装置. 三轴加速度计是测量X, Y, Z三个轴向的加速度, 包括运动引起的动态加速度和恒定的重力加速度
    - 物质组成
        - MEMS加速度计的核心物质是单晶硅
    - 基本原理: 牛顿第二定律F=ma + 电容变化检测
        - 惯性力作用: 传感器随载体做加速度运动时, 由于惯性, 悬浮的硅质量块会相对于固定的衬底产生位移
        - 位移转电容
        - 电容转电压
        - ADC
    - 种类
        - 压阻式
            - 原理: 加速度使质量块扭曲相连的压敏电阻
            - 特点: 结构简单, 温漂严重
        - 电容式
            - 原理: 加速度改变质量块于固定电机间的电容
            - 特点: 灵敏度高, 功耗低, 温度稳定性好
        - 压电式
            - 原理: 加速度使压电材料产生形变, 从而输出电荷
            - 特点: 动态范围大, 频率响应宽, 适合检测高频振动和冲击. 不能测量静态加速度(重力)
        - 热对流式
            - 原理: 加热一个微小气腔内的其他, 热气流在加速度作用下发生偏移, 由热电偶检测
            - 特点: 无任何质量块, 抗冲击力强, 可靠性高
        - 隧道电流式
            - 原理: 加速度使极尖与电极间距变化, 导致隧道电流剧变
            - 特点: 灵敏度极高, 但制造款男, 噪声大
        - 谐振式
            - 原理: 加速度改变谐振梁的应力, 从而改变其谐振频率, 通过测量频率变化得出加速度
            - 特点: 输出为准数字信号(频率), 抗干扰能力强, 精度高


### MPU6050概述：六轴运动传感器
    - 轴: x, y, z
    - 
1. **主要组成和功能**
    - **三轴MEMS速率陀螺仪传感器**
    - **三轴MEMS加速度计传感器**
    - 数字运动处理器（DMP）引擎
    - 主I2C以及SPI串行通信接口
    - 辅I2C串口接口: 用于连接第三方磁力计及其他传感器
    - 时钟模块
    - 传感器数据寄存器
    - FIFO缓存
    - 中断控制器
    - **数字输出温度传感器**
    - 陀螺仪和加速度计自检功能
    - 偏置调节与低压差线性稳压器
    - 电荷泵
2. **应用场景**
    - 无人机/四轴飞行器
        - 实时监测飞行姿态。
    - 智能手环/穿戴设备
        - 计步、手势识别。
    - 机器人平衡控制
        - 如两轮平衡车。
    - 手机/游戏手柄
        - 屏幕旋转、体感交互。
    - 工业设备
        - 振动监测、倾斜检测。

### MPU6050 I2C通信简介
    - S: 起始信号
    - AD: SlaveAddress
    - W: 写标志
    - R: 读标志
    - RA: Register Address
    - DATA: 数据
    - P: 停止信号
1. **单字节写入时序**
```
Master S            AD+W            RA              DATA    P
Slave       ACK             ACK             ACK
```
2. **突发写时序**: 批量写
```
Master S            AD+W            RA              DATA            DATA            P
Slave       ACK             ACK             ACK             ACK             ACK
```    
3. **单字节读取时序**
```
Master S       AD+W      RA       S        AD+R                 NACK    P
Slave    ACK        ACK      ACK     ACK         ACK     DATA    
```
4. **突发读取时序**
```
Master S       AD+W      RA       S        AD+R                 ACK             NACK    P
Slave    ACK        ACK      ACK     ACK         ACK     DATA           DATA
``` 

### 寄存器简介
1. **电源管理**	
    - 地址范围: 0x6B, 0x6C
    - 功能: 复位、睡眠、时钟源、轴使能
2. **采样率与配置**	
    - 地址范围: 0x19, 0x1A	
    - 功能: 采样分频、数字低通滤波器
3. **传感器配置**	
    - 地址范围: 0x1B, 0x1C	
    - 功能: 陀螺仪量程、加速度计量程
4. **中断配置**	
    - 地址范围: 0x37, 0x38, 0x3A	
    - 功能: 中断引脚、中断使能、中断状态
5. **数据输出**	
    - 地址范围: 0x3B - 0x48	
    - 功能: 加速度计、温度、陀螺仪数据
6. **FIFO**	
    - 地址范围: 0x23, 0x72, 0x73, 0x74	FIFO 
    - 功能: 使能、使能状态、读写操作
7. **I2C/SPI 控制**	
    - 地址范围: 0x6A, 0x24, 0x25	
    - 功能: 用户控制、主 I2C 配置
8. **身份与自检**	
    - 地址范围: 0x75, 0x0D - 0x12	
    - 功能: 设备 ID、传感器自检
9. **偏移校准**	
    - 地址范围: 0x06, 0x13 - 0x17, 0x77	
    - 功能: 偏移寄存器
10. **运动检测**	
    - 地址范围: 0x1F - 0x22, 0x69, 0x6D - 0x71	
    - 功能: 运动/自由落体/零运动检测
